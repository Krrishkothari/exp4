# Calculator Application

This is a simple calculator application demonstrating Git branching and merging.

## Features
- Addition (add.py)
- Subtraction (sub.py)
- Multiplication (mul.py)

## Usage
Run `python main.py` to test all calculator operations.

## Git Workflow Demonstrated
- Feature branch development
- Branch merging
- Version tagging

## Note
This project demonstrates proper version control practices using Git.
